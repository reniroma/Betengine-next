# BetEngine Next.js App Scaffold

A clean Next.js 14 App Router project with Tailwind CSS, ESLint, and Prettier pre-configured. The
project uses the `src/` directory convention and includes starter directories for components, shared
libraries, and styles.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app`: App Router entry points, layouts, and pages.
- `src/components`: Shared UI components.
- `src/lib`: Reusable utilities and application logic.
- `src/styles`: Additional style modules.
- `public`: Static assets served at the root.

Lint and formatting scripts are available via `npm run lint` and `npm run format`.
